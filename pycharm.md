
| 快捷键                                        | 作用                                   |
| --------------------------------------------- | -------------------------------------- |
| `ctrl + j`                                 | 显示可用的代码模板                     |
| `ctrl + b`                                 | 查看函数、类、方法的定义               |
| `ctrl + space`                                | 万能代码提示快捷键，一下不行按两下     |
| `ctrl + alt + l`                           | 格式化代码                             |
| `alt + enter`                                 | 万能代码修复快捷键                     |
| `ctrl + /`                                    | 注释/反注释代码                        |
| `shift双击`                               | 万能搜索快捷键                         |
| `ctrl + d` / `command + y`                 | 复制/删除一行代码                      |
| `ctrl + shift + -` / `ctrl + shift + +` | 折叠/展开所有代码                      |
| `F2`                                          | 快速定位到错误代码                     |
| `ctrl+ alt + F7`                           | 查看哪些地方用到了指定的函数、类、方法 |
